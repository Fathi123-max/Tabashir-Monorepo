Key Details

  • Dev Dependency

    Add flutter_launcher_icons: ^0.13.1 to dev_dependencies

  • Config Block

    Add flutter_launcher_icons: section to pubspec.yaml with platform-specific configs

  • Command

    dart run flutter_launcher_icons — generates all icons from source

  • Android Adaptive

    Foreground = logo with transparent background. Background = solid #0F4C81

  • Git

    Generated icons stay git-ignored. Source logos and pubspec config are committed.
