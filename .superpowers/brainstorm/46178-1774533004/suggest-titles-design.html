Onboarding Wizard with AI Suggestions

A smart onboarding flow that uses lightweight AI to assist the user without long wait times.

The Optimized Wizard Flow

Wizard Steps (5 Total)
1
Resume Upload: User selects PDF/DOCX.
Immediate call to /suggest-job-titles (very fast).
2
Target Roles: User sees Suggested Roles extracted from their CV. They can select these or add others.
3
Locations: Select preferred cities/countries.
4
Personal Details: Confirm Nationality and Gender.
5
Final Review & Processing: User clicks "Finish" → Full AI matching begins.

Technical Detail: /suggest-job-titles

  • Request: Multipart file (The resume).
  • Response: {"success": true, "suggested_job_titles": ["Full Stack Developer", "Software Engineer", ...]}
  • Implementation: Triggered automatically when Step 1 completes. Pre-populates the chips in Step 2.

Why this is better?

  • Reduced Friction: User doesn't have to type their job titles from scratch.
  • AI Value: Shows the power of Tabashir early in the onboarding.
  • Accuracy: Guides the user toward roles that the system actually found in their CV.
