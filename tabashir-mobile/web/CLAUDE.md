# CLAUDE.md

This directory contains the Flutter web build output for the Tabashir mobile app.

## Directory Contents

- `index.html` - Main HTML entry point for the web app
- `manifest.json` - PWA manifest configuration
- `favicon.png` - Application favicon
- `icons/` - App icons in various sizes (192x192, 512x512)
- `flutter_bootstrap.js` - Generated during build (contains Flutter runtime)

## Important Notes

This is a **build output directory**. Do not edit these files manually. They are generated by the Flutter build process from the source code in `lib/`.

## Building for Web

To rebuild the web version:

```bash
cd /Users/Apple/Documents/tabashir/tabashir-mobile

# Build for web
flutter build web --release

# Build with specific base href
flutter build web --release --base-href /tabashir/

# Serve locally for testing
flutter run -d chrome
```

The build output will be in either:
- `build/web/` (default build output)
- `web/` (this directory, depending on build configuration)

## Web-Specific Configuration

The `index.html` template uses `$FLUTTER_BASE_HREF` placeholder, which gets replaced during build. Set the base href using `--base-href` flag during build.

## Deployment

To deploy this web app:

1. Build the project: `flutter build web --release`
2. Upload contents of `build/web/` to your web server
3. Configure server to serve `index.html` for all routes (SPA behavior)
4. For GitHub Pages: use `--base-href /repository-name/`
